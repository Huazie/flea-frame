#Intellij IDEA 快捷键

1. Ctrl + Alt + T       选中代码块，自动导入（Surround with）
2. Shrift + F6          重命名（Rename）
3. Ctrl + Y             删除一行
4. Ctrl + Alt + L       格式化代码
5. Alt + Insert         自动生成代码（Generate）
6. Ctrl + Q             查看方法说明
7. Ctrl + Shrift + F10  运行（Run）
8. Alt + F7             查看引用
9. Ctrl + F             单个类查找
10. Ctrl + Shift + F     全局查找
11. Ctrl + R             单个类查找替换
12. Ctrl + Shift + R     全局查找替换
